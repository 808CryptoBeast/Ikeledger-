# Credential and Badge Model

Credentials prove learning activity and verification status.
They do not represent ownership of cultural knowledge.

## Proof-of-learning record example

{
  "user": "public_profile_id",
  "wallet": "rXXXXXXXXXXXXXXXXXXXX",
  "lesson_id": "DT-01",
  "lesson_title": "The Songlines - Stories That Are Maps",
  "realm": "Dreamtime",
  "completed_at": "2026-05-22T00:00:00Z",
  "mana_awarded": 13,
  "verification_status": "verified"
}

## Badge metadata fields

- Badge name
- Description
- Source lesson or module
- Date earned
- Issuer
- Wallet address or profile ID
- Verification hash
- Revocation status
