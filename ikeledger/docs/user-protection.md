# User Protection Guidelines

## Consent standards

Users must be informed before every wallet action:

- Connect wallet
- Sign message
- Submit transaction
- Claim reward
- Link or unlink wallet

## Risk labels

- Safe: Read-only account lookup
- Low Risk: Wallet connection
- Medium Risk: Message signing
- High Risk: Mainnet payment flow
- Blocked: Unknown or secret-related input

## Mainnet handling

Mainnet must show explicit warning text: real assets may be involved.
Testnet remains the default network in learning mode.
